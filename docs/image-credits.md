# Image credits

Every image used in the KavaWorks demo is listed here with its source, photographer (where known), and license. AI-generated imagery is not used for Pacific cultural content.

Artist avatars are illustrated SVG silhouettes filled with each artist's nation pattern — no photographic portraits of specific real individuals are used. Fictional personas are identified in the seed data with the `avatarStyle: 'illustrated-silhouette'` marker.

## Work photography

All work images used in the marketplace demo are typographic placeholder cards rendered client-side using the cultural pattern library (see the `<defs>` block in `KavaWorks.html`). No external photography of specific artworks is embedded — this sidesteps rights-clearance questions for a local-only demo.

## Cultural pattern SVGs

All 14 nation patterns and 5 theme patterns used throughout the app are inline SVG `<pattern>` definitions authored specifically for this demo. They use geometric cultural vocabulary (chevrons, diamonds, interlocking triangles, weaving lattices) — no representational motifs, no sacred figures.

The author of the SVG pattern set acknowledges that these simplified geometric forms are inspired by — not copies of — the living traditions of siapo, tīvaevae, masi, hiapo, kapa, ngatu, bilum, and kōwhaiwhai. Where KavaWorks is deployed beyond this demo, pattern approval by cultural practitioners and elders is required.
